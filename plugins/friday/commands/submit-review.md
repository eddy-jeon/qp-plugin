# 리뷰 결과 PR 코멘트 게시

현재 세션에서 진행된 리뷰 결과를 PR 코멘트로 게시합니다.

## 워크플로우

### 1. 리뷰 내용 수집

현재 세션에서 `/friday:review-pr`로 진행한 리뷰 내용을 수집합니다. 리뷰 항목이 없으면 사용자에게 알리고 종료합니다.

### 2. 포맷 변환

review-formatter agent를 사용하여 리뷰 내용을 PR 코멘트용 포맷으로 변환합니다.

### 3. 미리보기

변환된 리뷰 내용을 사용자에게 보여주고 승인을 요청합니다.

AskUserQuestion 도구를 사용합니다:

**질문: 이 내용으로 PR 코멘트를 게시할까요?**
- 옵션: "게시", "수정 후 게시", "취소"

- **게시**: 바로 PR 코멘트를 게시합니다
- **수정 후 게시**: 사용자가 수정 사항을 알려주면 반영 후 다시 미리보기를 보여줍니다
- **취소**: 게시를 취소하고 종료합니다

### 4. PR 코멘트 게시

승인 시 `gh` CLI로 코멘트를 게시합니다:

```bash
gh pr comment {pr번호} --body "$(cat <<'EOF'
{포맷팅된 리뷰 내용}
EOF
)"
```

### 5. 원래 브랜치로 복귀

```bash
# 원래 브랜치로 돌아가기
git checkout {ORIGINAL_BRANCH}

# stash한 변경사항이 있으면 복원
git stash pop
```

### 6. 결과 보고

- 게시된 PR 코멘트 URL
- 복귀한 브랜치 정보
