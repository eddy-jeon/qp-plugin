# Session Learner

Claude Code 세션에서 학습 내용을 자동으로 추출하여 지정된 Git 레포지토리에 저장하는 플러그인입니다.

## 기능

- 세션에서 배운 개념, 해결한 문제, 사용한 기술 등을 구조화된 형식으로 추출
- 지정된 레포지토리에 `data/{YYYY-MM-DD}/{uuid}.md` 형식으로 저장
- 세션 종료 시 자동 저장 (Stop 훅)
- 수동 저장 커맨드 제공

## 설치

```bash
# 플러그인 디렉토리로 Claude Code 실행
claude --plugin-dir /path/to/session-learner

# 또는 settings.json에 추가
```

## 사용법

### 1. 초기 설정

먼저 학습 기록을 저장할 Git 레포지토리 경로를 설정합니다:

```
/session-learner:setup
```

설정 프롬프트에서 레포지토리의 절대 경로를 입력하세요.

### 2. 수동 저장

현재 세션의 학습 내용을 수동으로 저장하려면:

```
/session-learner:save-session
```

### 3. 자동 저장

세션 종료 시 유의미한 학습 내용이 있으면 자동으로 저장됩니다.

## 저장되는 내용

```markdown
# 세션 학습 기록

**날짜**: 2026-01-19
**세션 ID**: abc123-def456
**프로젝트**: my-project

## 요약
세션에서 수행한 주요 작업 요약

## 학습 내용
- 배운 개념이나 기술

## 해결한 문제
- 해결한 버그나 이슈

## 사용한 기술/패턴
- 적용한 도구, 라이브러리, 패턴

## 핵심 인사이트
- 중요한 깨달음이나 팁

## 참고 코드
관련 코드 스니펫

## 관련 파일
- 수정하거나 참고한 파일 목록
```

## 설정 파일

설정은 `.claude/session-learner.local.md`에 저장됩니다:

```yaml
---
repository_path: /path/to/your/learning-repo
---
```

## 레포지토리 구조

학습 기록이 저장되는 레포지토리 구조:

```
learning-repo/
└── data/
    ├── 2026-01-19/
    │   ├── abc123.md
    │   └── def456.md
    └── 2026-01-20/
        └── ghi789.md
```

## 컴포넌트

| 컴포넌트 | 설명 |
|----------|------|
| `/session-learner:setup` | 레포지토리 경로 설정 |
| `/session-learner:save-session` | 수동으로 학습 내용 저장 |
| `session-extractor` 에이전트 | 세션 분석 및 학습 내용 추출 |
| Stop 훅 | 세션 종료 시 자동 저장 트리거 |

## 주의사항

- 설정된 경로는 유효한 Git 레포지토리여야 합니다
- push 권한이 있어야 원격 저장이 가능합니다
- 민감한 정보(API 키, 비밀번호 등)는 자동으로 제외됩니다
- 유의미한 학습 내용이 없는 세션은 저장하지 않습니다

## 디버깅

자동 저장이 제대로 동작하는지 확인하려면:

```bash
claude --debug
```

훅 실행 로그를 확인할 수 있습니다.
